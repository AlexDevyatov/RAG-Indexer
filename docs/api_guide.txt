Руководство по API

====================
DEEPSEEK EMBEDDING API
====================

Эндпоинт: https://api.deepseek.com/v1/embeddings

Метод: POST

Заголовки:
- Authorization: Bearer sk-your-api-key
- Content-Type: application/json

Тело запроса:
{
    "model": "deepseek-embedding",
    "input": ["текст для эмбеддинга", "ещё один текст"]
}

Ответ:
{
    "data": [
        {"embedding": [0.1, 0.2, ...], "index": 0},
        {"embedding": [0.3, 0.4, ...], "index": 1}
    ],
    "model": "deepseek-embedding",
    "usage": {"prompt_tokens": 10, "total_tokens": 10}
}

Лимиты:
- Максимум 2048 токенов на запрос
- Максимум 100 текстов в одном запросе
- Rate limit: 100 запросов в минуту

====================
OLLAMA API
====================

Эндпоинт: http://localhost:11434/api/chat

Метод: POST

Тело запроса:
{
    "model": "llama3.2",
    "messages": [
        {"role": "system", "content": "Системный промпт"},
        {"role": "user", "content": "Вопрос пользователя"}
    ],
    "stream": true
}

Параметры генерации:
- temperature: 0.7 (креативность)
- top_p: 0.9 (ядерная выборка)
- max_tokens: 2048 (максимальная длина ответа)

Доступные модели:
- llama3.2 (3B параметров, быстрая)
- llama3.2:1b (1B параметров, очень быстрая)
- mistral (7B параметров, качественная)
- codellama (для кода)

Команды Ollama:
- ollama pull <model> — скачать модель
- ollama list — список моделей
- ollama run <model> — запустить модель
- ollama serve — запустить сервер

====================
FAISS
====================

FAISS (Facebook AI Similarity Search) — библиотека для эффективного поиска похожих векторов.

Основные индексы:
- IndexFlatL2 — точный поиск по L2 расстоянию
- IndexFlatIP — точный поиск по скалярному произведению
- IndexIVFFlat — приблизительный поиск с кластеризацией
- IndexHNSW — граф для быстрого поиска

Пример использования:
```python
import faiss
import numpy as np

# Создание индекса
dimension = 1536
index = faiss.IndexFlatL2(dimension)

# Добавление векторов
vectors = np.random.random((100, dimension)).astype('float32')
index.add(vectors)

# Поиск
query = np.random.random((1, dimension)).astype('float32')
distances, indices = index.search(query, k=5)
```

Сохранение и загрузка:
```python
faiss.write_index(index, "my_index.faiss")
index = faiss.read_index("my_index.faiss")
```
